# 🃏 Blackjack Game (Vanilla JavaScript)

A modular Blackjack game built with HTML, CSS and Vanilla JavaScript using ES Modules.

The project simulates a simplified casino-style Blackjack experience with player vs dealer gameplay, score calculation, dealer logic and game state management.

---

## 🚀 Features

- 🎴 Deck generation and card shuffling
- 👤 Player vs Dealer gameplay
- 🧠 Blackjack detection
- 💥 Bust detection
- 🤖 Automatic dealer turn logic
- 📊 Dynamic score calculation with Ace handling
- 🧩 Modular JavaScript architecture
- 📱 Responsive layout

---

## 📁 Project Structure

```txt
js/
  core/
    game.js
    state.js

  logic/
    deck.js
    score.js
    result.js

  ui/
    ui.js
    elements.js

  shared/
    constants.js
    utils.js

  main.js
```

### Structure Overview

- `core/` → Main game flow and state management
- `logic/` → Game rules, score calculation and deck behavior
- `ui/` → DOM rendering and interface updates
- `shared/` → Shared constants and utility helpers
- `main.js` → Application entry point and event handling

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/kevbar98/blackjack-game.git
```

2. Open the project folder

```bash
cd blackjack-game
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm start
```
